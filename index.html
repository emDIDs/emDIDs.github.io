Welcome to our super amazing (and public) DID workspace.

This page demos some of the app parameters. See notes on GGB App Parameters page for more info.

The following 4 applets have the following parameters with their default settings:

  • language: ('en')
  • showZoomButtons: (false)
  • showLogging: (false)
  • capturingThreshold: (3)
  • enableUndoRedo: (true)
  • buttonShadows: (default not specified; assuming default is false)
  • buttonBorderColor: (Default is black, if the button background is white, otherwise one shade darker than the background color)
  • borderColor: (gray)
  • showAnimationButton: (default not specified; assuming default is true)
  • keyboardType: (default not specified; assuming default is "normal"; options are "scientific"|"normal"|"notes")
